HTML as the foundation of every website on the internet

Lets start with HTML markup tags.

Tags are keywords summarized by angle brackets.

The 'breaktag' is a self-closing tag used for breaking down long sentences.

The 'strong' tag is used to make a statement or sentence bold

The "em' tag is used to make a word or sentence italicized

the 'em' and 'strong' tag can be used together

The 'Img' tag is used to add/include image on the webpage

There are two types of lists; Ordered list and unordered list

Ordered list is used in sequential list format while unordered list are represented by bullets.

An example of ordered list is stated below;

  1. Greetings
  2. Hello world!
  3. Welcome
  4. Good day
  5. Good bye

Note that the list items must be between the body section

Form tags

Forms are used to collect data inputed by a user. They can be used as interface for web application. They tend to be used in conjunction with other programming langiuage to process the information inputed by the user. The basic tags used in the actual HTML of forms are form, input, text area, select and option.
